Color Legend

Green box

The selected primary person. This is the highest-confidence person detection and the one used for score, height ratio, area ratio, bottom ratio, and final validation.

Orange box

Other person detections found during low-threshold analysis. These are not the selected primary person. In production we should only reject if a second person is large enough, not just because an orange box exists.

Candidate Rules

primary_person_score >= 0.50

RT-DETR must be reasonably confident the selected primary object is a person.

height_ratio >= 0.40

The selected person box must cover at least 40% of the image height, so the person is not tiny in the background.

area_ratio >= 0.12

The selected person box must cover at least 12% of the full image area, so the person is a meaningful subject.

bottom_ratio >= 0.60

The selected person box must extend at least 60% down the image. This rejects face-only/shoulder-only crops while allowing non-full-body photos.

blur_score >= 45

The image must not be extremely blurry. This threshold is permissive because clean studio photos can have lower edge scores.
